# VUE CLI3 多页面 demo

### 主要内容：
* elementUI
* cli3 多页面
* 多页面下的单页面（SPA）路由配置

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
